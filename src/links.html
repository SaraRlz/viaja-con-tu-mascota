Fuentes y recursos utilizados

En esta página se recogen todas las fuentes de información, imágenes y recursos utilizados en la elaboración del proyecto “Viaja con tu mascota”. Se incluyen créditos, licencias y atribuciones cuando corresponde.

Contenido textual

Las descripciones de los destinos han sido elaboradas manualmente a partir de información pública de:

  • Portal oficial de turismo de España
  • Turismo de Asturias
  • Turismo de Galicia
  • Turismo de Islas Canarias

Imágenes utilizadas

Todas las imágenes utilizadas en el proyecto provienen de bancos de imágenes libres de derechos o con licencia apta para uso académico de Unsplash

Herramientas del proyecto

  • Editor usado: VS Code
  • Preprocesador CSS: SASS / SCSS
  • Dependencias instaladas:
    • baguettebox.js – Librería js para galerías de imágenes en lightbox.
  • Navegadores probados: Chrome, Firefox
